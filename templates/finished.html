Naraka AI

This project was in sync with a linear algebra class I was taking at the time where we learned many different methods to solving systems of equations with matrixes. In this particular example I wanted to explore the correlational data in a video game I played between pro teams, possibly even see if there was rationale for causation so that I could learn from it in my own play. Amidst the design I realized it would be beneficial to abstract alot of these methods in case either I found more features to add to the matrixes or I wanted to reuse the code for other projects. I ended up creating a small framework to help analyze statistics using Least Square's Approximation to varying polynomial degrees.
